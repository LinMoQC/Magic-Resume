Edit without fear — Magic Resume keeps a **version history**, so you can review and restore an earlier version anytime.

## Where to find it

The top of the editor has a **Version Management / Version History** entry. Open it to see this resume's past versions, each labeled with its **last update time**.

<Steps>
<Step title="Open version history">
Click **Version History** at the top.
</Step>
<Step title="Review or restore">
Select an older version to view it; confirm to roll back to that one. ⚠️ To verify visually: the exact restore button depends on the panel.
</Step>
</Steps>

<Note>
**Is there history in local mode too?** Yes — history is kept locally as well. With [cloud sync](/en/guide/cloud-sync) on, all versions are backed up to the cloud and visible across devices. With none, it shows "No version history".
</Note>

<Tip>
Before a big change (like a full [tailor-to-a-job](/en/guide/ai-optimize) or [translation](/en/guide/ai-translate)), relax — if you don't like the result, roll back from version history in one click.
</Tip>
