**设置**（左侧栏 **设置**）里主要三块：模型配置、云端同步、个人访问令牌。

## 模型配置（自带 Key 用 AI）

AI（AI Lab、AI PDF 导入）在后端跑、需要**登录**（云端模式）。想用自己的 API Key 代替平台额度，就在 **设置 → 模型配置** 里填：

- **API 提供商**（OpenAI / Anthropic / Gemini 等）
- **API 密钥**
- **模型**
- **Base URL**、**最大令牌数**（在"高级设置"里）

填完点 **测试连接**，看到 **连接成功** 就说明能用了，状态会变 **AI 已就绪**。

<Note>
**密钥只保存在你的浏览器本地，不会上传。** 这也是自托管/本地模式下用 AI 的方式——见 [部署到 Vercel](/zh/getting-started/vercel)。
</Note>

## 云端同步

在 **设置 → 云端同步** 里开关同步，详见 [云端同步](/zh/guide/cloud-sync)。

## 个人访问令牌（PAT）

想让 **Claude Code / Cursor** 这类 AI 工具直接读改你的简历？得先建一个**个人访问令牌**：

<Steps>
<Step title="创建令牌">
**设置 → 个人访问令牌 → 创建**。生成一串 `mr_pat_…` 开头的值。
</Step>
<Step title="马上复制保存">
这串值**只显示一次**。把它当密码一样保管——它能读写你所有简历。
</Step>
</Steps>

<Warning>
PAT 等于你简历的钥匙。**别贴到聊天、别提交进代码仓库。** 泄露了就回这里**吊销**它，再建一个新的。
</Warning>

接着就能 [接入 AI 编程工具](/zh/guide/mcp) 了。
