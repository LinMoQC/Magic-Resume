打开任意一份简历就进入编辑器。它分成两半：**左边是表单（你填），右边是实时预览（立刻看到效果）**。左边动一下，右边马上变。

## 填内容：按分区来

简历由一个个**分区**组成，从上到下常见的有：

| 分区 | 放什么 |
|---|---|
| **基本信息** | 姓名、联系方式（邮箱、电话、所在地、个人主页） |
| **个人总结** | 一段话概括你是谁、想找什么 |
| **工作经历** | 公司、职位、时间、做了什么 |
| **教育经历** | 学校、专业、学历、时间 |
| **项目经历** | 项目名、你的角色、成果 |
| **技能 / 语言 / 证书** | 技能标签、语言能力、证书资质 |

<Tip>
经历/项目的**描述框支持富文本**——能加粗、分点列表。多用「动词 + 成果 + 数字」，比如"主导支付重构，接口耗时从 800ms 降到 120ms"。
</Tip>

## 每个分区能做的操作

在一个分区里，把鼠标移到某个条目上（或点它的菜单），你能：

- **新增**：加一条新经历/项目/技能。
- **编辑**：改这条的内容。
- **复制**：基于这条复制一份，改改就是新的。
- **移除**：删掉这条。
- **显示 / 隐藏**：点眼睛图标，把某条临时藏起来（不删，只是不出现在简历上）。⚠️待视觉核对：不同岗位投不同版本时很好用。

<Note>
**改了会不会丢？** 不会。编辑器**自动保存**到你当前浏览器；登录并开启云同步后，还会自动同步到云端（顶部会显示 **已同步 / 同步中 / 待同步**）。
</Note>

## 顶部工具栏

编辑器顶部/侧边有一排按钮（图标 + 文字）：

- **AI 助手** —— 打开 [AI Lab](/zh/guide/ai-lab)，让北极星（Polaris）帮你写/改/分析。
- **分享** —— 生成分享链接，让别人看并留批注（需登录，见 [分享与批注]）。
- **反馈** —— 给我们提建议 / 报 bug。
- **导出 PDF** —— 一键把当前简历导成 PDF 下载，见 [预览与导出](/zh/guide/export)。
- **放大 / 缩小 / 重置缩放** —— 调预览的显示大小。
