做简历、导出 PDF **免费**、不用账户。**AI（AI Lab、AI PDF 导入）需要登录** —— 可以自带 API Key 免费用，或订阅用平台额度（不用自己配 Key）并解锁更多。

## 免费 vs 付费

<CardGroup cols={2}>
  <Card title="免费档" icon="gift">
    体验核心能力：完整编辑器、模板、导出，加上一定的每日/每周 AI 生成次数与可用模型。
  </Card>
  <Card title="付费档（Pro）" icon="crown">
    解锁全部能力：更多（乃至不限）每日/每周生成次数、更多可用模型、核心模型。
  </Card>
</CardGroup>

## 怎么升级

在账户菜单点 **升级到 Pro**（或 **升级套餐**），选一个档位完成订阅。订阅成功会提示 **订阅已创建**。你的**剩余额度**随时能在账户里看到。

<Note>
**档位、价格、功能由后台配置，可能随时调整**——以 App 里升级页显示的为准。部分能力可能标注"即将推出"。
</Note>

<Tip>
不订阅也能用 AI——**先登录**，再到 [设置 → 模型配置](/zh/guide/settings) 填**你自己的 API Key**（由后端使用，代替平台额度；密钥只存浏览器本地）。
</Tip>
