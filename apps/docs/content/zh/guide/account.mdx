Magic Resume **不登录也能用**——做简历、导出 PDF 都不需要账户。那什么时候该注册？

## 要不要登录？

<CardGroup cols={2}>
  <Card title="不登录（本地模式）" icon="laptop">
    简历只存在**当前浏览器**里，只有你能看到。适合快速做一份就走。缺点：换设备/清缓存就没了，**没有 AI**、不能分享、不能多端。
  </Card>
  <Card title="登录（云端模式）" icon="cloud">
    简历存到云端，并解锁 **AI 功能**（AI Lab、AI PDF 导入）：还有**多设备同步**、自动备份所有版本、生成**分享链接**收集批注。
  </Card>
</CardGroup>

## 怎么注册 / 登录

<Steps>
<Step title="打开登录">
点右上角的账户头像（或访问登录页）。你会看到 **登录 Magic Resume**。第一次用就点 **注册**。
</Step>

<Step title="选一种方式">
支持三种，任选其一：

- **用 Google 继续**
- **用 GitHub 继续**
- **用邮箱继续**（输入邮箱 → 收验证码 / 设密码）

<Tip>用 Google / GitHub 最省事，一键就好，不用记新密码。</Tip>
</Step>

<Step title="开启云同步">
登录后**云同步默认不是开的**——你的数据要不要上云，由你决定。到 **设置 → 云端同步**，打开 **启用云端同步**（会先给你看一段数据说明）。开启后，这台设备的改动就会自动同步到云端。
</Step>
</Steps>

<Note>
**已经在本地做了简历，登录会丢吗？** 不会。本地已有的简历还在；开启云同步后它们会被同步上云。
</Note>

## 账户里还能做什么

登录后点账户头像，能看到：**个人资料**、**设置**、**语言**切换、**退出登录**；以及你的**订阅**用量（当前计划、本期还剩多少次 AI）。想解锁全部 AI 能力可 **升级到 Pro**（见 [订阅与计费]，⚠️部分能力可能标注"即将推出"）。

<Warning>
云端模式需要后端服务在线。用官方 [magic-resume.cn](https://magic-resume.cn) 时一切就绪；如果是**自己部署**的，云同步/AI 需要你把后端也部署起来——那属于技术活，见 [开发手册](/zh/introduction)。
</Warning>
