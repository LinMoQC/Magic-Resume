---

右侧的**实时预览**就是你简历最终的样子——所见即所得。满意了就导出 PDF。

## 导出 PDF

<Steps>
<Step title="点「导出 PDF」">
在编辑器工具栏点 **导出 PDF** 按钮。它会**在你本地生成 PDF**（提示"正在本地生成 PDF..."），完成后自动下载。
</Step>
<Step title="拿到文件">
导出成功会提示 **PDF 导出成功**，文件就在你浏览器的下载目录里。这份 PDF 可以直接投递。
</Step>
</Steps>

<Note>
**PDF 在你本地生成**——不上传到服务器。所以断网也能导，你的内容不经过我们的后端。
</Note>

<Tip>
导出前用**放大 / 缩小**把预览拉到 100% 通读一遍，检查有没有内容溢出到第二页、日期对齐之类的小问题。
</Tip>

## 也能导出 JSON

除了 PDF，你还能把简历导成 **JSON**（编辑器里的 **JSON** 面板 → **下载 JSON**）。JSON 适合：

- **备份**：存一份，换电脑/换浏览器时 [导入](/zh/guide/create-import) 回来。
- **搬运**：在两份简历之间搬内容。

<Warning>
没登录时你在**本地模式**，简历只存在当前浏览器。**投递前最好先导一份 JSON 备份**，或者 [登录开启云同步](/zh/guide/account)，免得清缓存后白做。
</Warning>
